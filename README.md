# Home server

Documentation and configuration for a **local-first** home server: media, personal file sync, photo/camera storage, and local voice/AI—accessible remotely in a **safe** way (VPN-first, not “open the whole LAN to the internet”).

This repository holds **plans, notes, and eventually** `docker-compose` files, config templates, and scripts so everything can be **versioned, repeatable, and customized** to your setup.

- **Full roadmap and principles:** [docs/plan.md](docs/plan.md)
- **Hardware we’re planning around:** older Intel i5, **8 GB RAM**, **1 TB** HDD, **Linux** (Ubuntu Server or Debian LTS are fine defaults)

When the GitHub repo is private again, the same content stays in your clone; public was only to simplify first push and collaboration during bootstrap.
